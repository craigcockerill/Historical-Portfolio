Colophon

This site was developed using Sublime Text 2. On a Retina Macbook Pro.

Type is set in Bernini Sans & Herb by Just Another Foundry. All type is hosted via Typekit.

The site is powered by Statamic content management system. All CSS was precompiled along with Bourbon & Neat using SASS.

  • Hosted via A Small Orange
  • Version controlled using Bitbucket and Tower
  • Deployed via SSH using DeployHQ

Process

This site was designed largely in the browser. After sketching and ideation I constructed a half wireframe, half mockup in Illustrator.

Given the power of Statamic with the nature of my own portfolio I'm able to design and iterate rapidly with a fully functioning site.

After a heavy iterative development process, the site was tested and finessed across modern browsers.
