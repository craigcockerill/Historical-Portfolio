Colophon

This site was developed using Sublime Text 2. On a Retina Macbook Pro.

Type is set in Bernini Sans & Herb by Just Another Foundry. All type is hosted via Typekit.

The site is powered by Statamic content management system. All CSS was precompiled along with Bourbon & Neat using SASS.

  • Version controlled using Bitbucket and Tower
  • Deployed via SSH using DeployHQ
